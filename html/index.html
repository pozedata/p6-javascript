Règles du jeu

Part un beau temps, 2 joueurs décident d'aller faire une partie de pistolet à eau dans le jardin, le but est simple, battre l'autre !

Le terrain de jeu

Voici quelques spécificités du terrain

    - Le jardin est une grille de 10X10
    - Des buissons apparaîtrons et seront infranchissables
    - Les joueurs seront placés a l'opposé l'un de l'autre
    - Au sol, il y auras 4 armes que les joueurs pourront ramasser
    - Il peut y avoir jusqu'à 4 types d'armes différentes

Sur les cotés du terrain, se trouve les interfaces joueurs :

    - La case du haut represente le joueur
    - En dessous se trouvent les caractéristiques du joueur
    - Ensuite, la case pour l'instant vide, contiendra le déroulement du jeu
    - Et enfin les boutons liées au joueur

Liste des armes

Pistolet a eau : arme de base du joueur, 10 points de dégats

Pistolet laser : 15 points de dégats

Fusil à pompe : 20 points de dégats

Lance grenade : 25 points de dégats

Fusil d'assaut : 30 points de dégats

Déroulement du jeu

Le jeu se déroule en 2 étapes:

    - l'étape de mouvement
    - l'étape de combat

L'étape de mouvement

Chaques joueurs diposent de 3 points de mouvement par tour qu'il peut disposer comme il le souhaite. Les mouvements possible sont représentés par les cases vertes autour de lui. Le joueur peut utiliser le bouton fin de tour pour terminer son tour de jeu sans utiliser tous ses déplacements.

Quand un joueur ramasse une arme, il dépose l'arme précédemment equipée.

L'étape de mouvement prend fin lorsque les joueurs se retrouvent sur des cases adjacentes

L'étape de combat

Lors de l'etape de combat, les joueurs ne peuvent plus se déplacer. Il peuvent attaquer ou se défendre.

Attaquer permet d'infliger des dégats.

Se défendre permet de reduire de 50% les dégats de l'adversaire

Le jeu se termine quand un des joueurs se retrouve à 0 de santé
