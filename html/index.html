Règle du jeu

Par un beau temps, 2 joueurs décide d'aller faire une partie de pistolet a eau dans le jardin, le but est simple, battre l'autre !

Le terrain de jeu

Voici quelques spécificités du terrain

    - Le jardin est une grille de 10X10
    - Des buisson apparaitrons et seront infranchissable
    - Les joueurs seront placer a l'opposé l'un de l'autre
    - Au sol, il y auras 4 armes que les joueurs pourront ramasser
    - Il peut y avoir jusqu'a 4 type d'armes différentes

Sur les cotés du terrain, se trouve les interfaces joueurs :

    - La case du haut represente le joueur
    - En dessous se trouve les caractéristiques du joueur
    - Ensuite, la case pour l'instant vide contiendras le déroulement du jeu
    - Et enfin les bouton liée au joueur

Liste des armes

Pistolet a eau : arme de base du joueur, 10 points de dégats

Pistolet laser : 15 points de dégats

Fusil a pompe : 20 points de dégats

Lance grenade : 25 points de dégats

Fusil d'assaut : 30 points de dégats

Déroulement du jeu

Le jeu se déroule en 2 étapes:

    - l'étape de mouvement
    - l'étape de combat

L'étape de mouvement

Chaque joueurs dipose de 3 point de mouvement par tour qu'il peut disposer comme il le souhaite. Les mouvement possible sont représenter par les case vertes autour de lui. Le joueur peut utiliser le bouton fin de tour pour termner son tour de jeu sans utiliser tous ses déplacement.

Quand un joueur ramasse une arme, il dépose l'arme précédemment equipé.

L'étape de mouvement prend fin lorsque les joueur se retrouve sur des cases adjacentes

L'étape de combat

Lors de l'etape de combat, les joueurs ne peuvent plus se déplacer. Il peuvent attaquer ou se défendre.

Attaquer permet d'infliger des dégats.

Se défendre permet de reduire de 50% les dégats de l'dversaire

Le jeu se termine quand un des joueur se retrouve a 0pdv
